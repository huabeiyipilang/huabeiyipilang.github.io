杀进程，根据进程名

adb shell busybox killall [process name]

例如：adb shell busybox killall com.android.phone

发送按键信息

adb shell input keyevent [keycode] (keycode详见[KeyEvent.java][1])

例如：adb shell input keyevent 82 (发送menu键)

安装APP

adb install [apk path]

例如：adb install c:kankan.apk

卸载APP

adb uninstall [package name]

例如：adb uninstall com.xunlei.kankan.tv

清空应用数据

adb shell pm clear [package name]

例如：adb shell pm clear com.xunlei.kankan.tv

打开APP

adb shell am start -n [package name]/[class name]

例如：adb shell am start -m com.xunlei.kankan.tv/com.kankan.tv.StartupActivity
